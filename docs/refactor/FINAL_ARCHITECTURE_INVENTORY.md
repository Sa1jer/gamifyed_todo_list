## Implemented Boundaries

- `lib/analytics/`: immutable scalar analytics and weekly view data.
- `lib/coordinators/`: Task, completion, reward, RoadMap, Skill/Goal, and
  Review/session mutation policy.
- `lib/persistence/`: save scheduler, snapshot store/codec, migration policy,
  legacy codec/domain access, and local preference storage.
- `lib/models/`: eight persisted declaration modules; `models.dart` is only a
  compatibility export barrel.
- `lib/presentation/`: deterministic prepared data for high-cost screens.
- `lib/widgets/main_page/desktop_*`: ordinary desktop shell, sidebar, main
  workspace, right rail, quest row, selected header, and support modules.
- `lib/widgets/weekly_analytics/`, `progress_hub/`, and `tasks/`: cohesive
  feature sections with explicit inputs/callbacks and local lifecycle owners.

## Remaining Concentrations

- AppState is still the public cross-domain facade and owns startup, device
  side effects, final notification, and persistence sequencing.
- RoadMap's painter/editor workspace remains a deliberate `part` library.
- The largest remaining presentation files are bounded but still deserve
  evidence-led review when their responsibilities next change.
- Real-Hive process termination, disk-full behavior, and native memory traces
  cannot be established by source structure alone.
