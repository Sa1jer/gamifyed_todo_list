# Repository Codex procedures

These prompt files are repository-local reusable procedures. They intentionally
reference [AGENTS.md](../../../AGENTS.md) and nearby workflow templates rather
than duplicating project context. Use them in order: `explore-task`,
`implement-task`, `review-diff`, `fix-reviewed-findings`, and
`verify-acceptance`; use `fix-bug` for a focused defect workflow.
